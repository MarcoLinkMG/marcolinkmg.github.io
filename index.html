¿Que aprendimos en cada Desafio?

En este desafío validamos nuestros conocimientos en HTML
  • Crear un página web utilizando HTML.
  • Manejar las etiquetas p, h1, h2, h3, a, img, title y meta.
  • Implementar la estructura assets del proyecto revisada en la lectura.
  • Emplear adecuadamente la estructura y sintaxis de las etiquetas de un documento HTML, para dar solución a una problemática.

Se aplicaron las técnicas aprendidas en clases y en guias para poder construir un landing page
  • Implementar CSS utilizando archivos hojas de estilo externas al documento HTML.
  • Homogeneizar el diseño del sitio.
  • Estructurar una página en términos y assets del proyecto.
  • implementar bordes, margen, padding y selectores de etiqueta o clase según sea necesario.

En este desafío donde pusimos a prueba nuestros conocimientos en las propiedades de display y float, html semántico.
  • Estructuracion del proyecto.
  • Ubicar horizontalmente los elementos, utilizando las propiedades display o float.
  • Implementación de Iconos con Font Awesome.
  • Utilizar etiquetas semánticas para definir las distintas secciones de la página.

En este desafío aplicamos nuestros conocimientos sobre el framework bootstrap.
  • Bootstrap nos permite tener una versión para escritorio, para tablet y para teléfonos móviles.
  • Se utilizaron las clases de Bootstrap que asignan margin ypadding acorde a la maqueta del desafío.
  • Se diiseñaron Cards de los cupones utilizando como base el componente Card de Bootstrap.
  • Se Utilizo la grilla de Bootstrap para distribuir los objetos Cards
